# Age Calculator

## 📖 Project Description

This Python program calculates a user's age in different time units. After entering the age in years, the user can choose a unit (months, weeks, days, hours, minutes, or seconds), and the program converts the age to the selected unit and displays the result. This project demonstrates the use of user input, conditional statements, arithmetic operations, and string formatting in Python.

## ✨ Features

- Accepts the user's age in years.
- Converts the age into:
  - Months
  - Weeks
  - Days
  - Hours
  - Minutes
  - Seconds
- Allows the user to select the desired time unit.
- Displays the calculated result in the selected unit.

## 🛠️ Technologies Used

- Python 3

## 📚 Concepts Used

- `input()`
- `if` / `elif`
- Arithmetic Operations
- `strip()`
- f-strings
