# Name Formatter

## 📖 Project Description

This Python program asks the user to enter their first, middle, and last name. It then formats the names by removing extra spaces, capitalizing the first letter of each name, and displaying the middle name as an initial. Finally, it prints a friendly greeting with the formatted full name.

## ✨ Features

- Accepts the user's first, middle, and last name.
- Removes leading and trailing spaces.
- Capitalizes the first letter of each name.
- Displays the middle name as an initial.
- Prints a personalized greeting.

## 🛠️ Technologies Used

- Python 3

## 📚 Concepts Used

- `input()`
- `strip()`
- `capitalize()`
- String slicing (`:.1s`)
- Formatted strings (f-strings)

## ▶️ Example

### Input

```text
What's your first name? ahmed
What's your middle name? mohamed
What's your last name? ali
```

### Output

```text
Hello Ahmed M Ali Happy to see you
```
