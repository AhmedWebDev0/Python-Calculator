# 🎨 ASCII Name Art

A simple Python project that transforms a user's name into beautiful ASCII art using **pyfiglet** and displays it in a user-selected color using **termcolor**.

## ✨ Features

* Convert any name into ASCII art.
* Choose your favorite text color.
* Display colorful text directly in the terminal.
* Easy to understand and beginner-friendly.

## 📦 Requirements

Install the required libraries:

```bash
pip install pyfiglet termcolor
```

Or, if you use the Python launcher on Windows:

```bash
py -m pip install pyfiglet termcolor
```

## ▶️ How to Run

Run the program with:

```bash
py your_file.py
```

## 💻 Example

```text
Please Your Name: Ahmed
What is your favorite color? cyan
```

Output:

```text
    _    _                          _
   / \  | |__  _ __ ___   ___  __| |
  / _ \ | '_ \| '_ ` _ \ / _ \/ _` |
 / ___ \| | | | | | | | |  __/ (_| |
/_/   \_\_| |_|_| |_| |_|\___|\__,_|
```

*(Displayed in the selected color.)*
