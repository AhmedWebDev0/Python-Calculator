# 🎬 Short If (Conditional Expressions)

A simple Python project that demonstrates how to use conditional statements (`if`, `elif`, `else`) and the **short if (ternary operator)** to determine whether a user is old enough to watch a movie.

---

## 📌 Features

* 🎥 Defines the minimum movie age rating.
* 👤 Accepts the user's age.
* ✅ Checks if the user is allowed to watch the movie.
* ⚡ Demonstrates both the standard `if` statement and the Python ternary operator (short if).

---

## 🛠️ Technologies Used

* Python 3

## ▶️ How It Works

1. Set the minimum required age for the movie.
2. Get the user's age.
3. Compare the user's age with the movie rating.
4. Display the appropriate message using:

   * `if / elif / else`
   * Short If (Ternary Operator)
